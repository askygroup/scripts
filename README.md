# scripts
脚本库



#### 脚本存放规范

##### 目录：

​	1、按功能区分，一类功能一个目录，一种功能一个子目录

##### 文件：

​	1、必要的说明性文件

​		每个目录中必须有一个说明性文件，文件内容包括：

​		1）当前目录下，文件功能介绍，脚本的功能，其他文件作用

​		2）每个脚本迭代纪录信息，新增了什么功能，优化了什么方式，修改了什么内容

​		3）脚本的逻辑思路(可选)

​	2、脚本版本命名

​		每次的版本迭代，脚本都需要保留：

​		1）版本：name-1.*

​		2）子版本：name-1-1.*
