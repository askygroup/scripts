# 逐行加载 ASCII 艺术字文件
读取 ASCII 艺术字文件，然后逐行打印，每打印一行等待一会儿，形成逐行加载的效果


---


#### 脚本迭代记录

##### loading_ascii.sh

|脚本迭代版本|描述|
|---|---|
|loading_ascii.sh|读取 ASCII 艺术字文件，然后逐行打印，每打印一行等待0.1秒|
|||

##### loading_ascii.py

|脚本迭代版本|描述|
|---|---|
|loading_ascii-1.py|读取 ASCII 艺术字文件，然后逐行打印，每打印一行等待0.1秒|
|loading_ascii-2.py|读取 ASCII 艺术字文件，然后逐行打印，加载的总等待时间为1秒|
|||
